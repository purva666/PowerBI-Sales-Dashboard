## Project Overview
This project is an interactive Sales Dashboard developed using Microsoft Power BI. It provides insights into sales performance, customer behavior, product performance, and business trends through interactive visualizations.

## Tools Used
- Microsoft Power BI
- Microsoft Excel / CSV
- Data Modeling
- DAX
- Power Query
  ## Dataset
The dashboard is built using the following datasets:
- Customers
- Orders
- Products

## Dashboard Features
- Sales Performance Analysis
- Customer Insights
- Product Performance
- Interactive Filters and Slicers
- KPI Cards
- Charts and Visualizations
